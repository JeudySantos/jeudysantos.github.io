Lista de Etiquetas HTML

# Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
1 <html> Define la raíz del documento HTML. <html>...</html> Contiene las etiquetas <head> y <body>.
2 <head> Contiene metadatos e información sobre el documento. <head>...</head> Incluye etiquetas como <title>, <meta>, <link>, etc.
3 <title> Define el título del documento que se muestra en la pestaña del navegador. <title>Mi página web</title> Se encuentra dentro de la etiqueta <head>.
4 <meta> Proporciona metadatos sobre el documento, como la codificación de caracteres, palabras clave, descripción, etc. <meta charset="UTF-8"> Se utiliza para mejorar el SEO y la accesibilidad.
5 <link> Enlaza hojas de estilo externas (CSS) u otros recursos. <link rel="stylesheet" href="style.css"> Se encuentra dentro de la etiqueta <head>.
6 <style> Define estilos CSS dentro del documento HTML. <style>body { background-color: #f0f0f0; }</style> Se puede incluir dentro de <head> o <body>.
7 <body> Contiene el contenido visible de la página web. <body>...</body> Incluye todas las etiquetas que se muestran al usuario.
8 <h1> a <h6> Encabezados de diferentes niveles (h1 es el más importante, h6 el menos importante). <h1>Título principal</h1> Se utilizan para estructurar el contenido y mejorar el SEO.
9 <p> Define un párrafo de texto. <p>Este es un párrafo de ejemplo.</p> Se utiliza para organizar el texto en bloques.
10 <br> Inserta un salto de línea. Este es un texto <br> con un salto de línea. No tiene etiqueta de cierre.
11 <hr> Crea una línea horizontal. <hr> Se utiliza para separar secciones de contenido. No tiene etiqueta de cierre.
12 <a> Define un enlace o hipervínculo. <a href="https://www.ejemplo.com">Enlace</a> El atributo 'href' especifica la URL del enlace.
13 <img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción de la imagen"> Los atributos 'src' especifica la ruta de la imagen y 'alt' proporciona una descripción alternativa. No tiene etiqueta de cierre.
14 <div> Define una división o contenedor genérico. <div>...</div> Se utiliza para agrupar elementos y aplicar estilos CSS.
15 <span> Define un tramo o contenedor en línea. <span>...</span> Se utiliza para aplicar estilos CSS a una parte específica de texto.
16 <ul> Define una lista no ordenada (con viñetas). <ul>...</ul> Contiene elementos de lista <li>.
17 <ol> Define una lista ordenada (con números). <ol>...</ol> Contiene elementos de lista <li>.
18 <li> Define un elemento de lista. <li>Elemento 1</li> Se utiliza dentro de <ul> o <ol>.
19 <dl> Define una lista de descripción. <dl>...</dl> Contiene términos <dt> y descripciones <dd>.
20 <dt> Define un término en una lista de descripción. <dt>Término</dt> Se utiliza dentro de <dl>.
21 <dd> Define la descripción de un término en una lista de descripción. <dd>Descripción del término</dd> Se utiliza dentro de <dl>.
22 <table> Define una tabla. <table>...</table> Contiene filas <tr> y celdas <td> o <th>.
23 <tr> Define una fila en una tabla. <tr>...</tr> Contiene celdas <td> o <th>.
24 <th> Define una celda de encabezado en una tabla. <th>Encabezado</th> Se utiliza para los encabezados de columnas o filas.
25 <td> Define una celda de datos en una tabla. <td>Dato</td> Contiene el contenido de una celda.
26 <form> Define un formulario HTML para la entrada de datos del usuario. <form>...</form> Contiene elementos de formulario como <input>, <select>, <textarea>, etc.
27 <label> Define una etiqueta para un elemento de formulario. <label for="nombre">Nombre:</label> Asocia una etiqueta de texto con un elemento de formulario.
28 <input> Crea un campo de entrada para datos del usuario. <input type="text" name="nombre"> Puede ser de varios tipos: texto, contraseña, email, etc.
29 <select> Define una lista desplegable. <select>...</select> Contiene opciones <option>.
30 <option> Define una opción en una lista desplegable. <option value="valor">Opción</option> Se utiliza dentro de <select>.
31 <textarea> Define un área de texto multilínea. <textarea name="mensaje"></textarea> Se utiliza para ingresar texto largo.
32 <button> Define un botón. <button type="submit">Enviar</button> Puede ser de varios tipos: submit, reset, button.
33 <fieldset> Agrupa elementos de formulario relacionados. <fieldset>...</fieldset> Se utiliza para organizar visualmente los formularios.
34 <legend> Define una leyenda para un grupo de elementos de formulario (<fieldset>). <legend>Información personal</legend> Se utiliza para describir el grupo de campos.
35 <code> Define un fragmento de código. <code>int x = 10;</code> Se utiliza para mostrar código de programación.
36 <pre> Define texto preformateado. <pre>Texto preformateado</pre> Mantiene los espacios y saltos de línea.
37 <samp> Define una salida de ejemplo de un programa. <samp>Resultado: 10</samp> Se utiliza para mostrar ejemplos de salida.
38 <kbd> Define texto que debe ser ingresado por el usuario. <kbd>Ctrl + C</kbd> Se utiliza para indicar teclas o comandos.
39 <var> Define una variable. <var>x</var> Se utiliza para representar variables en matemáticas o programación.
40 <em> Define texto enfatizado. <em>Este texto está enfatizado</em> Se muestra en cursiva.
41 <strong> Define texto importante. <strong>Este texto es importante</strong> Se muestra en negrita.
42 <small> Define texto más pequeño. <small>Este texto es más pequeño</small> Se utiliza para texto secundario o notas al pie.
43 <del> Define texto eliminado. <del>Este texto ha sido eliminado</del> Se muestra tachado.
44 <ins> Define texto insertado. <ins>Este texto ha sido insertado</ins> Se muestra subrayado.
45 <u> Define texto subrayado. <u>Este texto está subrayado</u> Se utiliza para resaltar texto.
46 <i> Define texto en cursiva. <i>Este texto está en cursiva</i> Se utiliza para resaltar texto o para términos técnicos.
47 <b> Define texto en negrita. <b>Este texto está en negrita</b> Se utiliza para resaltar texto.
48 <mark> Define texto resaltado. <mark>Este texto está resaltado</mark> Se utiliza para resaltar texto importante.
49 <q> Define una cita corta. <q>Este es una cita corta</q> Se muestra entre comillas.
50 <blockquote> Define una cita larga. <blockquote>Este es una cita larga</blockquote> Se muestra con sangría.
51 <cite> Define el título de una obra (libro, película, canción, etc.). <cite>El Quijote</cite> Se muestra en cursiva.
52 <address> Define información de contacto. <address>Calle Falsa, 123</address> Se muestra en cursiva.
53 <abbr> Define una abreviatura o acrónimo. <abbr title="HyperText Markup Language">HTML</abbr> El atributo 'title' proporciona la forma completa.
54 <dfn> Define un término que se va a definir. <dfn>HTML</dfn> Se utiliza junto con <dd> en una lista de descripción.
55 <bdo> Define la dirección del texto (de izquierda a derecha o de derecha a izquierda). <bdo dir="rtl">Este texto se muestra de derecha a izquierda</bdo> El atributo 'dir' especifica la dirección.
56 <ruby> Define anotaciones ruby para texto en idiomas asiáticos. <ruby>漢<rt>かん</rt></ruby> Se utiliza para mostrar la pronunciación de caracteres.
57 <rt> Define la pronunciación de un carácter ruby. <rt>かん</rt> Se utiliza dentro de <ruby>.
58 <rp> Define texto que se muestra si el navegador no admite anotaciones ruby. <rp>(</rp> Se utiliza dentro de <ruby>.
59 <time> Define una fecha y/u hora. <time datetime="2023-10-27">27 de octubre de 2023</time> El atributo 'datetime' especifica la fecha y/u hora en formato ISO 8601.
60 <date> Define una fecha. <date>27 de octubre de 2023</date> Se utiliza para mostrar fechas.
61 <datetime> Define una fecha y hora. <datetime>2023-10-27 10:00</datetime> Se utiliza para mostrar fechas y horas.
62 <meter> Define una medida o rango. <meter value="50" min="0" max="100">50/100</meter> Se utiliza para mostrar progreso o niveles.
63 <progress> Define una barra de progreso. <progress value="50" max="100"></progress> Se utiliza para mostrar el progreso de una tarea.
64 <output> Define el resultado de un cálculo. <output>Resultado</output> Se utiliza en formularios para mostrar resultados.
65 <details> Define detalles que el usuario puede ver u ocultar. <details>...</details> Se utiliza para mostrar información adicional que no es necesaria de inmediato.
66 <summary> Define un encabezado para los detalles (<details>). <summary>Más información</summary> Se muestra como un encabezado que el usuario puede hacer clic para mostrar u ocultar los detalles.
67 <nav> Define una sección de navegación. <nav>...</nav> Se utiliza para agrupar enlaces de navegación.
68 <aside> Define contenido relacionado pero independiente del contenido principal. <aside>...</aside> Se utiliza para barras laterales o información adicional.
69 <article> Define un artículo independiente en un documento. <article>...</article> Se utiliza para entradas de blog, noticias o contenido similar.
70 <section> Define una sección genérica en un documento. <section>...</section> Se utiliza para agrupar contenido relacionado.
71 <header> Define la cabecera de un documento o sección. <header>...</header> Contiene información introductoria.
72 <footer> Define el pie de página de un documento o sección. <footer>...</footer> Contiene información de contacto, derechos de autor, etc.
73 <main> Define el contenido principal de un documento. <main>...</main> Debe haber un solo elemento <main> por documento.
74 <figure> Define contenido independiente y auto-contenido (imágenes, diagramas, etc.). <figure>...</figure> Puede tener una leyenda con <figcaption>.
75 <figcaption> Define una leyenda para una figura (<figure>). <figcaption>Descripción de la imagen</figcaption> Se utiliza dentro de <figure>.
76 <audio> Define contenido de audio. <audio controls>...</audio> Se utiliza para insertar audio en una página web.
77 <video> Define contenido de video. <video controls>...</video> Se utiliza para insertar video en una página web.
78 <source> Define múltiples recursos de medios para <audio> y <video>. <source src="video.mp4" type="video/mp4"> Se utiliza para especificar diferentes formatos de video o audio.
79 <track> Define pistas de texto para <video> y <audio>. <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español"> Se utiliza para subtítulos, leyendas u otros tipos de pistas de texto.
80 <embed> Define contenido externo (aplicaciones o plugins). <embed src="plugin.swf" type="application/x-shockwave-flash"> Se utiliza para incrustar contenido multimedia o interactivo.
81 <object> Define contenido incrustado (imágenes, audio, video, aplicaciones, etc.). <object data="imagen.svg" type="image/svg+xml"></object> Se utiliza para incrustar diferentes tipos de contenido.
82 <param> Define parámetros para <object>. <param name="autoplay" value="true"> Se utiliza para configurar el comportamiento del objeto incrustado.
83 <canvas> Define un área para gráficos dibujados mediante scripts (JavaScript). <canvas></canvas> Se utiliza para crear gráficos, animaciones o juegos.
84 <svg> Define gráficos vectoriales escalables (SVG). <svg>...</svg> Se utiliza para crear gráficos que se pueden escalar sin perder calidad.
85 <math> Define fórmulas matemáticas. <math>...</math> Se utiliza para mostrar ecuaciones y expresiones matemáticas.
86 <noscript> Define contenido alternativo para usuarios con JavaScript desactivado. <noscript>Tu navegador no soporta JavaScript.</noscript> Se muestra si JavaScript no está habilitado.
87 <script> Define scripts (JavaScript). <script>...</script> Se utiliza para añadir interactividad a una página web.
88 <noscript> Define contenido alternativo para usuarios con JavaScript desactivado. <noscript>Tu navegador no soporta JavaScript.</noscript> Se muestra si JavaScript no está habilitado.
89 <template> Define contenido que se clonará y se insertará en el documento mediante JavaScript. <template>...</template> Se utiliza para crear plantillas reutilizables.
90 <slot> Define un marcador de posición para contenido que se proporcionará más adelante. <slot></slot> Se utiliza en componentes web.
91 <element> Define un nuevo elemento personalizado. <element name="mi-elemento">...</element> Se utiliza para crear elementos HTML personalizados.
92 <shadow> Define un árbol de sombra para un elemento. <shadow>...</shadow> Se utiliza en componentes web.
93 <content> Define el punto de inserción para el contenido distribuido. <content select=".nombre"></content> Se utiliza en componentes web.
94 <import> Importa recursos externos. <import href="componente.html"> Se utiliza para importar plantillas o componentes web.
95 <resource> Define un recurso que se puede utilizar en el documento. <resource type="text/css" href="estilos.css"> Se utiliza para incluir recursos como hojas de estilo o scripts.
96 <declare> Declara una variable. <declare name="variable">valor</declare> Se utiliza en lenguajes de marcado como XAML.
97 <overlay> Define una capa superpuesta. <overlay>...</overlay> Se utiliza para crear ventanas modales o elementos flotantes.
98 <dialog> Define un cuadro de diálogo. <dialog>...</dialog> Se utiliza para crear ventanas emergentes o cuadros de diálogo.
99 <menu> Define un menú. <menu>...</menu> Se utiliza para crear menús de navegación o menús contextuales.
100 <menuitem> Define un elemento de menú. <menuitem label="Opción 1"></menuitem> Se utiliza dentro de <menu>.
101 <command> Define un comando que se puede ejecutar. <command label="Guardar" onclick="guardar()"></command> Se utiliza para crear comandos en menús o barras de herramientas.
102 <keygen> Define un campo para generar claves. <keygen name="clave"> Se utiliza para crear claves públicas/privadas.
